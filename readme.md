# Rでふるさと納税のマップを眺める（Google cloudにデプロイするもの）

ふるさと納税の納税額や使用用途をマップ上に表示するShinyアプリです。Google cloudでは以下のアドレスに公開しています（2024/3現在）。
https://furusato-map-oxbrmwzuuq-dt.a.run.app
